# Robô tupiniquim

Os robôs estão em uma área escolhida para análise e curiosamente retangular e os robôs devem
andar por ela com suas câmeras ligadas coletando todo tipo de informação.
A posição de um robô é representada por uma combinação de coordenadas X
e Y e também uma letra representando a direção que ele está olhando. A área
é dividida em um grid para simplificar a navegação.

## Funções
- Você escolhe uma direção para se mover com os seguintes botões,"D" para virar a direita, "E" para virar a esquerda e "M" para se mover.
- O codígo é capaz de mostrar sua localização atual e se chegou ao limite do grid.
